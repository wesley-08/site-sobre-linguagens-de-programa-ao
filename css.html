inicio

CSS (Cascading Style Sheets) é uma linguagem usada para estilizar e formatar
documentos HTML ou XML. Com CSS, você pode definir a aparência visual de
elementos em uma página da web, como cores, fontes, espaçamento, layout
e muito mais. A principal função do CSS é separar a estrutura do conteúdo (que é
definida em HTML) do estilo ou apresentação da página.

Aqui estão algumas funcionalidades básicas do CSS:

1.Estilos de Texto: Definir fontes, tamanhos, cores e alinhamento.

p {
color: blue;
font-size: 16px;
text-align: center;
}

2.Layouts: Organizar o layout de uma página com grid, flexbox ou float.

.container {
display: flex;
justify-content: space-between;
}

3.Cores e Imagens de Fundo: Definir cores de fundo e imagens.

body {
background-color: lightgray;
background-image: url('imagem.jpg');
}

4.Espaçamento: Controlar o espaçamento interno (padding) e externo (margin) dos elementos.

div {
margin: 20px;
padding: 10px;
}

O CSS é essencial para tornar as páginas web visualmente atraentes e facilitar
a personalização de layout e design em diferentes dispositivos (responsividade).
